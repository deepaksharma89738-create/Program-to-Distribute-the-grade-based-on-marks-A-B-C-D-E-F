# Program-to-Distribute-the-grade-based-on-marks-A-B-C-D-E-F
This program is used to assign grades based on the marks obtained by a student. First, the user enters the marks. Then, using if-else conditions, the program checks the range of marks and assigns grades like A, B, C, D, E, or F. For example, higher marks get grade A, and lower marks get grade F. Finally, the program displays the grade.
